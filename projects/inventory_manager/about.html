Inventory Manager

A C++ Inventory Management Program designed to handle a collection of items with various attributes. This application includes item tracking, inventory persistence, and a graphical user interface (GUI) for ease of use.

Features

  • Item Management: Store multiple items with properties like name, description, image, and count.
  • Item Quantity Adjustments: Increase or decrease quantities, or delete items.
  • Inventory Persistence: Save and load inventory data from files, with a default sample provided.
  • Graphical User Interface (GUI): Intuitive interface with tabs for saving/loading inventories, item details, and help.

Planned Structure

Item Object

Each item will be stored as an object with attributes such as name, description, count, and image.

Functions

  • Add a new item to the inventory.
  • Delete an item from the inventory.
  • Adjust the quantity of an item.
  • Save or load inventory data from files.

GUI Implementation

  • Scrollable list of items with tabs for file operations, new inventory creation, and help.
  • Detailed view of items with options for editing or deletion.

Getting Started

Prerequisites

A C++ compiler (e.g., GCC, Clang, or MSVC) and a GUI framework (e.g., Qt, wxWidgets, or SFML).

Building the Project

  1. Clone the repository: 
    git clone https://github.com/Code-JL/Inventory-Manager
  2. Build the project using your preferred build system: 
    mkdir build
cd build
cmake ..
make

Future Enhancements
